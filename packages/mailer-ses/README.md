# Apoyo - Mailer SES

[![npm version](https://badgen.net/npm/v/@apoyo/mailer-ses)](https://www.npmjs.com/package/@apoyo/mailer-ses)

## Installation

Install peer dependencies:
`npm install @apoyo/std @apoyo/mailer`

Install package:
`npm install @apoyo/mailer-ses`
