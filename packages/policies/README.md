# Apoyo - Policies

[![npm version](https://badgen.net/npm/v/@apoyo/policies)](https://www.npmjs.com/package/@apoyo/policies)

## Installation

Install peer dependencies:
`npm install @apoyo/std`

Install package:
`npm install @apoyo/policies`
